
# 🌸 Flower Shop - Premium Floral Design Studio

A modern, responsive flower shop website built with React, TypeScript, and Tailwind CSS, integrated with Sanity CMS for content management.

## ✨ Features

- **Responsive Design**: Mobile-first approach with beautiful UI
- **Product Catalog**: Browse flowers by categories (Roses, Tulips, Lilies, etc.)
- **Service Pages**: Arrangements, Workshops, and Concierge services
- **Content Management**: Powered by Sanity CMS
- **Modern Stack**: React 19, TypeScript, Vite, Tailwind CSS

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
# Copy .env.local and update with your Sanity project details
VITE_SANITY_PROJECT_ID=ilxd7zjn
VITE_SANITY_DATASET=production
```

4. Start the development server:
```bash
npm run dev
```

5. Start Sanity Studio (in a separate terminal):
```bash
cd sanity
npm run dev
```
